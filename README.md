## Proyectos principales

### UniAvisos

Una aplicación Android diseñada para administrar y distribuir avisos mediante departamentos dinámicos, suscripciones y notificaciones.

El sistema integra:

* Departamentos administrados desde la nube
* Avisos cargados en tiempo real
* Suscripciones personalizadas
* Notificaciones mediante Firebase Cloud Messaging
* Panel provisional de administración
* Optimización de listeners y consultas
* Planeación de almacenamiento local, roles y escalabilidad

**Tecnologías**

`Kotlin` `Jetpack Compose` `Firebase Firestore` `Firebase Cloud Messaging`

### Pizzería Control ERP

Un sistema administrativo desarrollado para controlar el proceso completo de una pizzería.

No se limita a registrar productos.

Conecta inventario, recetas, órdenes, cocina, caja, entregas, proveedores, reportes y solicitudes de compra dentro de un mismo flujo.

El sistema incluye:

* Administración de inventario
* Productos, insumos y recetas
* Creación y seguimiento de órdenes
* Flujo de cocina
* Cobro en caja
* Descuento automático de existencias
* Registro de movimientos
* Proveedores y solicitudes de compra
* Reportes y pronósticos
* Envío supervisado de correos

**Tecnologías**

`PHP` `CodeIgniter` `MySQL` `HTML` `CSS` `JavaScript`

### CineBot

Un recomendador web de películas construido alrededor de una interfaz conversacional.

El usuario puede combinar géneros, emociones y otros filtros para recibir recomendaciones personalizadas.

Incluye:

* Flujo conversacional
* Filtros combinables
* Recomendaciones de tres películas
* Películas guardadas temporalmente
* Tarjetas interactivas
* Diseño responsivo
* Prototipado y análisis de experiencia de usuario

**Tecnologías**

`React` `JavaScript` `HTML` `CSS` `Figma`
